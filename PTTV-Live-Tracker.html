        How it works: Check an item on Active → moves to Archive (no lingering strikethrough on Active). Uncheck on Archive → restores. 📺 aired ( yellow marker-fill) = mentioned on-air; visible across Active and Archive so end-of-show you can see what made it vs what didn't. → todo ( orange flag) = pull into the end-of-session team-todos artifact. Per-item × and "Permanently delete all archived" are the only destructive operations — both confirm before acting. Both tabs start empty each session; the long-term project archive is the persistent store.
